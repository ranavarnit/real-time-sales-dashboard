# Real-Time Sales Analytics Dashboard

## Overview
Interactive sales analytics dashboard with **real-time KPIs** and automated data generation. Built with Streamlit and Plotly for executive decision-making and business intelligence.

## Live Demo
- Dashboard: `http://localhost:8501` (run locally)

## Features
- **Real-time KPIs**: Total Revenue, Orders, AOV, Unique Customers
- **Interactive Charts**: Pie (Category), Line (Trend), Bar (Region, Channel)
- **Date Range Controls**: Filter by Last 7/30/90 Days
- **Auto Data Generation**: 5,000+ sample records on load
- **Sub-second Performance**: Cached data processing

## Tech Stack
- Python, Streamlit
- Plotly, Pandas
- NumPy

## Quick Start
```bash
pip install streamlit plotly pandas numpy
streamlit run simple_sales_dashboard.py
```
